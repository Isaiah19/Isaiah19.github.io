Simple Shell

A simple UNIX command interpreter

Overview:
This project involves creating a simple UNIX command interpreter.
The shell is capable of reading commands from the standard input or from a file.

Purpose:
The purpose of this project is to understand the core concepts of
building a command-line interpreter, including process creation and management,
user input handling, and command execution.

Project Details

Objective:
The main objective of this project is to build a functional shell that can handle basic
commands and execute them like a typical UNIX shell.

Technologies Used:
C, Shell scripting.

Features:

  • Handles basic shell commands like `ls`, `pwd`, `echo`.
  • Includes built-in functions such as `cd`, `exit`.
  • Supports path execution for commands.

Challenges:
One of the challenges faced was implementing process creation and management,
as well as handling user input efficiently.

How to Use

Installation Instructions:

        git clone https://github.com/https://github.com/Isaiah19/simple_shell
        cd simple_shell
        gcc -Wall -Werror -Wextra -pedantic *.c -o hsh
        ./hsh

Usage:
To use the shell, run the executable and type commands just as you
would in a normal UNIX shell. For example:

        $ ./hsh
        ($) /bin/ls
        hsh main.c shell.c
        ($) exit

Conclusion

Lessons Learned:
During this project, I learned about process management in C,
how to handle user inputs, and the basics of building a shell.
It also reinforced my understanding of system calls and file I/O operations in UNIX.

Future Improvements:
In the future, I plan to add more built-in commands and improve error
